# Pr-ctica-Control-de-Datos
En este trabajo he hecho varios ejercicios en Java para comprobar datos que introduce el usuario por teclado.

En el primer ejercicio compruebo si una edad o un año son correctos usando condiciones if.

En el segundo ejercicio pido un PIN de 4 números y hago que el usuario lo escriba dos veces para comprobar que coinciden.

En el tercer ejercicio pido una fecha (día, mes y año) y compruebo si es válida, teniendo en cuenta los meses y los años bisiestos.
